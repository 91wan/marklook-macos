# Privacy

MarkLook renders files locally. It does not upload documents, does not collect analytics, and does not contact remote servers during preview rendering.

MarkLook does not include telemetry, accounts, cloud sync, or remote asset loading. The Quick Look preview extension is designed to read only the file URL handed to it by Quick Look.

Future changes that add any network access, telemetry, or document export behavior must be documented in a pull request and reviewed before merge.
